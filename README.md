TASK TRACKER - Nersisyan A.S.
#DESCRIPTION:
Task Tracker is a simple Django-based web application for managing daily tasks. It displays a list of tasks for the current day using Bootstrap styling.
#USAGE:
1. Click "Run" in Replit
2. Wait for the server to start
3. Open the Webview
4. The task list will be displayed on the main page
#VARIABLES YOU CAN CHANGE:
- Task list is located in templates/index.html
- Tasks are in <li class="list-group-item"> tags
- Modify the text inside these tags to change tasks
#REQUIREMENTS:
- Python 3.8+
- Django 4.0+
- Modern browser with JavaScript
